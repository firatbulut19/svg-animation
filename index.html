This website is designed and implemented by Fırat Bulut as a senior project for Bogazici University Computer Engineering. The project advisor is Haluk Bingöl, and the content is largely inspired by his earlier works in meraklisina.com. The animations are implemented using SVG (Scalable Vector Graphics) elements and JavaScript. Source code and a project poster can be reached from here.
